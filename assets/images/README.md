# Image Placeholders

This directory contains images for your portfolio. Please add the following images:

## Required Images

### Profile
- **profile.jpg** (500x500px recommended)
  - Your professional headshot or portrait
  - Format: JPG or PNG
  - Size: Keep under 500KB

### Projects
Place project screenshots in the `projects/` folder:

- **ecommerce.jpg** - E-Commerce Platform screenshot
- **task-app.jpg** - Task Management App screenshot
- **portfolio-designs.jpg** - Portfolio Designs showcase
- **dashboard.jpg** - Real-time Dashboard screenshot
- **cms.jpg** - Content Management System screenshot
- **api.jpg** - RESTful API visualization

**Recommended dimensions**: 1200x800px or 16:9 aspect ratio
**Format**: JPG or PNG
**Size**: Keep each under 300KB

### Favicon
- **favicon.png** (32x32px or 64x64px)
  - Your site icon
  - Format: PNG or ICO
  - Transparent background recommended

## Image Optimization Tips

1. **Compress images** before uploading
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Target: < 300KB per image

2. **Use appropriate formats**
   - Photos: JPG
   - Graphics/Logos: PNG
   - Icons: SVG (when possible)

3. **Responsive images**
   - Provide multiple sizes for different devices
   - Use srcset attribute for better performance

## Image Editing Tools

- **Online**: Canva, Figma, Photopea
- **Desktop**: Photoshop, GIMP, Affinity Photo
- **Compression**: TinyPNG, ImageOptim, Squoosh

## Current Placeholder Setup

The website is configured to work with placeholder images. Simply replace them with your actual images maintaining the same filenames.

## Need Help?

If you don't have project images yet, you can use:
- Screenshots from your GitHub repositories
- Design mockups from Figma/Adobe XD
- Placeholder services like Unsplash for temporary images
