# Profile Image Setup

Place your profile image here with the filename: `profile.jpg`

This image will appear in the hero section (blue square box) on the homepage.

Recommended specifications:
- Filename: profile.jpg
- Format: JPEG or PNG
- Dimensions: Square aspect ratio (e.g., 500x500, 800x800, or 1000x1000)
- File size: Under 500KB for optimal loading
- Type: Professional headshot or portrait photo

The image will be automatically styled with:
- Blue border matching the website accent color
- Rounded corners
- Responsive sizing for mobile devices
